Tab Drawer Test Page

This page is designed to test the Tab Drawer Chrome extension. You should see a drawer on the left side that pushes the page content to the right.

Test Actions:

Expected Behavior:

  • ✅ Drawer appears on the left side, pushing content to the right
  • ✅ Drawer shows all open tabs with favicons
  • ✅ Clicking tabs switches to them
  • ✅ Drawer can be resized by dragging the right edge
  • ✅ Drawer can be collapsed with the × button
  • ✅ Drawer appears on new tabs automatically

Test Scenarios:

  1. Open this page in a new tab - drawer should appear
  2. Open multiple tabs - all should show in the drawer
  3. Switch between tabs using the drawer
  4. Close tabs using the × button in the drawer
  5. Resize the drawer by dragging the right edge
  6. Collapse and expand the drawer
